Google Devfest 介绍

各位 Google Developers 们：

相约金秋，一年一度的谷歌开发者节 DevFest 再度来临。Android Wear、Android L、Polymer、 Material Design、移动开发、多屏适配……各种有趣的新技术、经验分享、开发案例，以及可以动手参与的 Codelab ，参加 DevFest 可以学习最新、最酷的技术，和技术专家面对面交流。参会者还将获得社区T恤、Google 贴纸和神秘礼物，作为一名开发者决不能错过这个免费的技术盛宴！

DevFest 就是开发者节的意思，是 Google 赞助发起的全球性开发者活动，也是 GDG 社区的年度活动。活动内容完全由世界各地的 GDG 社区志愿者自行策划组织，纯粹的技术分享活动，没有广告式的宣讲和厂商展位，过一个属于开发者自己的节日。

如今 Android 早已超越手机和平板的概念，无处不在。目前已经正式发布了3款 Android Wear 手表，还将延伸到电视和汽车等领域。

再好的产品没有好的设计和交互也难以被用户喜爱，Material Design 是最新的界面设计指南，适用于各种平台。要是你的产品还没有向 Material Design 设计方向转变，先机就会被竞争对手抓到。

下一代 Android 系统 Android L 即将正式发布，赶紧学习一下有什么新变化、新特性吧。

Web Components 是下一代 Web 技术标准，打破传统 Web 技术的各种限制，让开发者写出更加组件化、更加易于维护的代码。

Polymer 是基于 Web Components 新技术的 Web 框架，通过 Polyfill 让你在所有最新浏览器上使用这些新技术，利用 Paper UI 组件可以轻松开发出符合 Material Design 效果 Web 界面。

从手机到平板、笔记本、PC、电视，各种大小屏幕都要适配，那么问题来了，如何用最低成本适配这么多种不同大小的屏幕？学习响应式设计技术，只需额外一点工作即轻松解决跨平台多平台适配问题。

关于 GDG

GDG , 即 Google Developer Groups 的缩写（not 郭德纲）或 谷歌开发者社区，是谷歌开发者部门发起的全球性技术社区。 GDG 是面向对 Google 和开源、开放技术感兴趣的公益性开发者社区，内容涵盖 Web/Chrome、Android 和其它 Google API 等。全国已成立了18个 GDG 社区，赶快加入你身边的组织吧。

如何报名参加 DevFest ？访问中国谷歌开发者社区论坛 ChinaGDG.com ，然后在各地分论坛中找到 DevFest 活动通知。
